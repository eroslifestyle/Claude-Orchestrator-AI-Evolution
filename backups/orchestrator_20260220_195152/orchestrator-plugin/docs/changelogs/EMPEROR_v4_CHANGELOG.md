## COME USARE v4

```typescript
// Import principale
import { orchestrator, OrchestratorV4 } from './index-v4';

// Quick start
import { quickOrchestrate } from './index-v4';
const result = await quickOrchestrate([
    'Create GUI for user management',
    'Add database persistence',
    'Implement authentication'
]);

// Uso avanzato
const orch = new OrchestratorV4({
    maxConcurrent: 24,
    enableSmartModelSelection: true,
    enableAgentDiscovery: true
});

orch.addTask({ description: 'Complex security audit' });
orch.addTask({ description: 'Performance optimization' });

const stats = await orch.execute();
console.log(stats);
```
